# Python Lab Programs

This repository contains two simple Python programs:

## Files

### 1. Lab2.py
A basic calculator program that performs:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Exponentiation

### 2. trial.py
A simple user introduction program that:
- Accepts name, age, and hobby
- Displays a formatted introduction message
